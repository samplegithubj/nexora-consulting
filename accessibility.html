Accessibility Statement

Last Updated: March 2024

1. Our Commitment

Nexora ICT Consulting is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

2. Conformance Status

We aim to conform to the following standards:

  • WCAG 2.1: Web Content Accessibility Guidelines level AA
  • Section 508: U.S. federal accessibility standards
  • EN 301 549: European accessibility standards

3. Accessibility Features

Our website includes the following accessibility features:

3.1 Keyboard Navigation

  • All interactive elements are accessible via keyboard
  • Skip-to-content links
  • Logical tab order

3.2 Screen Reader Compatibility

  • Proper ARIA labels and roles
  • Semantic HTML structure
  • Alternative text for images

3.3 Visual Design

  • Sufficient color contrast (4.5:1 minimum)
  • Resizable text up to 200%
  • No information conveyed by color alone

3.4 Multimedia

  • Captions for videos
  • Transcripts for audio content
  • Controls for auto-playing media

4. Technical Specifications

Our website relies on the following technologies for accessibility:

  • HTML5
  • WAI-ARIA
  • CSS
  • JavaScript

5. Assessment Approach

We assess accessibility through:

  • Automated testing tools (axe, WAVE)
  • Manual testing by our team
  • User testing with people with disabilities
  • Regular accessibility audits

6. Known Limitations

Despite our best efforts, some content may not be fully accessible:

  • Third-party content we don't control
  • Legacy documents in PDF format
  • Complex data visualizations
